# Notebooks

The full modelling pipeline, exported from Google Colab. Run in order 01 -> 10.

| # | File | Stage |
|---|------|-------|
| 01 | 01_data_preprocessing.ipynb | Load, clean, structure data |
| 02 | 02_exploratory_analysis.ipynb | Exploratory data analysis |
| 03 | 03_feature_engineering.ipynb | Features + graph construction |
| 04 | 04_baseline_models.ipynb | Baseline models |
| 05 | 05_gnn_component.ipynb | Graph Neural Network |
| 06 | 06_rnn_component.ipynb | Recurrent Neural Network |
| 07 | 07_mem_integration.ipynb | Multidimensional Error Model |
| 08 | 08_combined_model.ipynb | Full GNN + RNN + MEM |
| 09 | 09_model_validation.ipynb | Validation + metrics |
| 10 | 10_results_and_ablation.ipynb | Results + ablation |

Export each from Colab via **File -> Save a copy in GitHub**, saving into this folder with the matching name.
